# Bexiter

A simple, fast note-taking app. Open, write, done.

## Quick Start

```bash
npm install
npm run dev
```

Open `http://localhost:3000` or `http://YOUR_IP:3000` on your local network.

## Features

- **WYSIWYG Editor** - Rich text, no markdown visible. Paste from anywhere.
- **Folders & Tags** - Organize your way
- **Search** - Find anything instantly
- **Collaboration** - Share notes with family/team (view/edit roles)
- **Dark/Light Theme**
- **Export** - Download as .md files
- **Admin Panel** - Manage users
- **3 Text Templates** - Compact / Medium / Reader

## Tech Stack

- **Frontend:** Next.js 14, React, TypeScript, Tailwind CSS
- **Editor:** TipTap (WYSIWYG)
- **Backend:** Supabase (PostgreSQL + Auth)
